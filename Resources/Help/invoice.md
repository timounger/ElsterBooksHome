# Rechnungen

Auf der Registerkarte `Einkommen` können Sie über den Button `Rechnung erstellen` Rechnungen in verschiedenen Formaten erzeugen.

## Rechnungsformate

- Excel (.xlsx): Nur für die manuelle Weiterverarbeitung geeignet. Rechnungen im Excel-Format sind nicht zulässig.
- XML: E-Rechnung im rein maschinenlesbaren Format.
- PDF: Klassische, ausschließlich menschenlesbare Rechnung. ⚠️ Keine gültige E-Rechnung – nur unter bestimmten rechtlichen Voraussetzungen zulässig.
- ZUGFeRD: Hybride E-Rechnung – für Mensch und Maschine lesbar.

## Rechnungsempfänger und Datenquelle

Der Rechnungsempfänger wird einfach über das Dropdown-Menü ausgewählt.

Die Daten stammen:

- beim Empfänger aus dem jeweiligen Kontakt
- beim Rechnungssteller aus den Unternehmensdaten in den Einstellungen

Für individuelle Anpassungen kann der „Erweiterte Modus“ aktiviert werden.

## Vorlagen

Rechnungsvorlagen können importiert und exportiert werden.
Das Format ist kompatibel mit dem Tool [PDF24 E-Rechnung erstellen](https://tools.pdf24.org/de/elektronische-rechnung-erstellen).

## Erstellung von PDF- und ZUGFeRD-Rechnungen

Für die Erzeugung von PDF- und ZUGFeRD-Dokumenten wird LibreOffice als portable Version benötigt.
Diese Version ist bereits korrekt vorkonfiguriert.

Alle benötigten Tools, einschließlich LibreOffice, können im Tab `Export` über den Button `Download Tools` heruntergeladen werden.
