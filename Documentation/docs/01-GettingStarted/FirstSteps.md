# Erste Nutzung & Einrichtung

## 1️⃣ Erste Anmeldung & Setup

- Starten Sie ElsterBooks.
- Beim ersten Start öffnet sich ein Dialog, in dem Sie Ihre **Unternehmensdaten** und **grundlegenden Einstellungen** hinterlegen können.

Siehe auch [➡️ Einstellungen](../05-Settings/CompanyInfo.md) für detaillierte Konfigurationen.

## 2️⃣ Externe Tools installieren

ElsterBooks nutzt externe Programme, die für bestimmte Funktionen erforderlich sind.  
Um alle Funktionen zu nutzen, müssen diese Tools nachgeladen werden.  

Im Tab **Export** können Sie über den Button **„Download Tools“** die benötigten Programme installieren.  
Die [Tools](https://github.com/timounger/ElsterBooksTools/releases/tag/latest) werden parallel zur Haupt-Exe abgelegt.

??? info "Verwendete Tools"
    - **LibreOfficePortable:** Erstellung von PDF- und ZUGFeRD-Rechnungen  
    - **PortableGit:** Versionskontrolle und Revisionierung der Daten  
    - **TortoiseGit:** Änderungsansicht beim Commit  
    - **Tesseract-OCR:** Texterkennung für die automatische Belegerfassung  
    - **plz.json:** Automatische Ortsergänzung anhand der PLZ  

## 3️⃣ Erste Buchung erstellen

Erfassen Sie Ihre ersten Daten, legen Sie Kunden an oder erstellen Sie direkt eine Rechnung.  

Weitere Details finden Sie unter [➡️ Belegerfassung](../02-Transactions/Transactions.md).
