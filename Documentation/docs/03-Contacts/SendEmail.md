# E-Mail senden

Mit ElsterBooks können Sie direkt aus der Anwendung heraus E-Mails an Ihre Kontakte versenden.  

- Durch einen **Doppelklick** auf die hinterlegte E-Mail-Adresse öffnet sich Ihr Standard-Mailprogramm.  
- Der **Empfänger**, **Betreff** und **Textvorlage** werden automatisch übernommen.  

Weitere Details zu den **Mailvorlagen** finden Sie hier: [➡️ Mailvorlage](../05-Settings/MailTemplate.md)
