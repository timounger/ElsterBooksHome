# Kontakt anlegen

Mit ElsterBooks können Sie Kundenkontakte einfach **anlegen und verwalten**, was die Rechnungsstellung und den E-Mail-Versand erheblich erleichtert.

## Details

- Es stehen zahlreiche Felder zur Verfügung, die den **E-Rechnungsanforderungen** entsprechen.  
- Die hinterlegten Kontaktdaten können direkt beim **Erstellen von Rechnungen** genutzt werden.  
- Durch die strukturierte Verwaltung behalten Sie jederzeit den Überblick über Ihre Geschäftspartner.
