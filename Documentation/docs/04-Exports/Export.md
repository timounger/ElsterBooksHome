# Export

## Übersicht

Erstellt eine **komplette Übersicht** aller erfassten Einnahmen und Ausgaben.  
Diese Funktion ermöglicht Ihnen eine schnelle Kontrolle und den Vergleich der Finanzdaten über den gesamten Zeitraum.

## Sicherung erstellen (Backup)

Erstellt ein **ZIP-Archiv** aller gespeicherten Dokumente und Buchungsdaten.  
Dies dient als **Sicherheitskopie**, die zur Wiederherstellung oder für externe Prüfungen verwendet werden kann.
