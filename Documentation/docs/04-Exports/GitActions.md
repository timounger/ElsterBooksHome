# Git-Aktionen

## Commit

Erstellt eine Versionierung des aktuellen Datenstands.  
Diese Funktion wird automatisch beim Beenden des Programms ausgelöst, sofern Änderungen an den Daten vorgenommen wurden.  

Sie ermöglicht die Nachverfolgung von Änderungen und stellt sicher, dass frühere Zustände bei Bedarf wiederhergestellt werden können.
