# Rechnungen erstellen

Auf der Registerkarte **Einnahmen** können Sie über den Button **„Rechnung erstellen“** Rechnungen in verschiedenen Formaten erzeugen.  

## Rechnungsformate

- **Excel:** Nur für interne oder manuelle Weiterverarbeitung geeignet. Nicht als offizielle Rechnung zulässig.  
- **XML:** Maschinell lesbare E-Rechnung für automatisierte Verarbeitung.  
- **PDF:** Klassische, ausschließlich menschenlesbare Rechnung. ⚠️ Nicht als elektronische E-Rechnung gültig – nur unter bestimmten rechtlichen Voraussetzungen zulässig.  
- **ZUGFeRD:** Hybride E-Rechnung, die sowohl von Menschen als auch von Maschinen gelesen werden kann.

## Rechnungsempfänger und Datenquelle

Der Rechnungsempfänger wird über ein **Dropdown-Menü** ausgewählt.  

Die Daten stammen aus:

- **Empfänger:** dem ausgewählten Kontakt  
- **Rechnungssteller:** den Unternehmensdaten, die in den Einstellungen hinterlegt sind  

Für individuelle Anpassungen kann der **Erweiterte Modus** aktiviert werden.

## Vorlagen

Rechnungsvorlagen können **importiert und exportiert** werden.  

Das Format ist kompatibel mit dem Tool [PDF24 E-Rechnung erstellen](https://tools.pdf24.org/de/elektronische-rechnung-erstellen).

## Erstellung von PDF- und ZUGFeRD-Rechnungen

Für die Generierung von PDF- und ZUGFeRD-Dokumenten wird **LibreOffice Portable** benötigt.
