# Sonstige Dokumente

Im Tab **Dokumente** können Sie Unterlagen erfassen, die keine direkten Zahlungsvorgänge betreffen, z. B. Verträge, Schriftverkehr oder andere relevante Belege.  

> Hinweis: Es können ausschließlich **PDF-Dateien** hinzugefügt werden.

## Metadaten

Für jedes Dokument lassen sich folgende Metadaten hinterlegen:  

- **Datum:** Dokumentendatum  
- **Name:** frei wählbarer Titel oder Bezeichnung des Dokuments  

Diese Metadaten erleichtern die strukturierte Ablage und die spätere Suche nach Dokumenten innerhalb von ElsterBooks.
